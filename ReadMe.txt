========================================================================
    콘솔 응용 프로그램 : MonitoringServer 프로젝트 개요
========================================================================

 TCP 서버, Web 서버가 잘 동작하고 있는지
 모티터링 하기위하여 만들어진 서버입니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더(PCH) 파일 MonitoringServer.pch와 미리 컴파일된 형식(PCT) 파일 StdAfx.obj를 빌드하는 데 사용됩니다.

/////////////////////////////////////////////////////////////////////////////
기타 참고:

응용 프로그램 마법사에서 사용하는 "TODO:" 주석은 사용자가 추가하거나 사용자 지정해야 하는 소스 코드 부분을 나타냅니다.
